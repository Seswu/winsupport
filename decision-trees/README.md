## Using the Trees

Each tree follows a consistent format:

```
Branch condition?
├─ YES → Action or sub-question
│         └─ RESULT → Resolution step
└─ NO  → Alternative path
          └─ RESULT → Resolution step
```

Terminal actions (in **bold**) are concrete steps you take at the machine or in an admin portal.

Cross-references to other trees are formatted as `[see 01-01](01-01_account-auth_password-reset.md)`.
