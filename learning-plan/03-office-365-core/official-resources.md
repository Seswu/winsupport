# Official Resources: Office 365 Core

## Microsoft Learn Paths

### 1. Manage your Microsoft 365 tenant
- **URL:** https://learn.microsoft.com/en-us/training/paths/manage-your-microsoft-365-tenant/
- **Topics covered:** Tenant management, Microsoft 365 Apps deployment, admin roles, tenant health and services
- **Relevance:** Very High — directly applicable to supporting Office apps
- **Surprise factor for Linux users (1-5):** 3 — Cloud-based tenant management has no Linux equivalent
- **Estimated time:** ~2 hours

### 2. Outlook troubleshooting documentation
- **URL:** https://learn.microsoft.com/en-us/outlook/troubleshoot/
- **Topics covered:** Outlook connectivity issues, profile problems, AutoDiscover troubleshooting, OST/PST issues
- **Relevance:** Very High — Outlook is the most-supported Office app
- **Surprise factor for Linux users (1-5):** 3 — AutoDiscover architecture and profile management have no Linux equivalent
- **Estimated time:** ~45 minutes
- **Notes:** Reference documentation rather than a training module; bookmark for troubleshooting

### 3. Troubleshoot Microsoft 365 Apps
- **URL:** https://learn.microsoft.com/en-us/troubleshoot/microsoft-365-apps/
- **Topics covered:** Activation issues, update problems, performance issues, installation errors, repair procedures
- **Relevance:** Very High — practical troubleshooting focus
- **Estimated time:** ~40 minutes
- **Notes:** Reference documentation; bookmark the activation and update sections specifically

### 4. Use OneDrive in Microsoft 365
- **URL:** https://learn.microsoft.com/en-us/training/modules/m365-onedrive-collaboration-use/
- **Topics covered:** OneDrive features, sharing, coauthoring, sync app, Files On-Demand, Known Folder Move
- **Relevance:** Very High — OneDrive issues are daily tickets
- **Surprise factor for Linux users (1-5):** 3 — Files On-Demand and KFM concepts are unique
- **Estimated time:** ~38 minutes
