# Official Resources: Microsoft 365 Admin & Licensing

## Microsoft Learn Paths

### 1. Describe Microsoft 365 apps and services
- **URL:** https://learn.microsoft.com/en-us/training/paths/describe-microsoft-365-core-services-concepts/
- **Topics covered:** M365 vs. Office 365 differences, service overview, licensing model, admin centers
- **Relevance:** Very High — foundational knowledge for all admin tasks
- **Surprise factor for Linux users (1-5):** 3 — The bundled nature of M365 (Windows + Office + Security) is different from Linux subscription models
- **Estimated time:** ~1 hour

### 2. Manage users and groups
- **URL:** https://learn.microsoft.com/en-us/training/modules/manage-users-groups/
- **Topics covered:** Active Directory, Microsoft Entra ID, user and group management, security and distribution groups, authentication
- **Relevance:** Very High — daily admin tasks
- **Estimated time:** ~40 minutes

### 3. Describe Microsoft 365 pricing, licensing, and billing options
- **URL:** https://learn.microsoft.com/en-us/training/modules/identify-licensing-options-available-microsoft-365/
- **Topics covered:** Pricing models, billing management, base vs. add-on licensing, subscription options
- **Relevance:** Very High — licensing issues are common support tickets
- **Estimated time:** ~25 minutes

### 4. Manage tenant health and services in Microsoft 365
- **URL:** https://learn.microsoft.com/en-us/training/modules/manage-tenant-services-microsoft-365/
- **Topics covered:** Service health monitoring, usage analytics, network connectivity assessments, incident response, Microsoft support requests
- **Relevance:** Very High — first thing to check for multi-user issues
- **Estimated time:** ~49 minutes
