# Official Resources: Diagnostic Tools & Reference

## Sysinternals Documentation

### 1. Sysinternals Suite Overview
- **URL:** https://learn.microsoft.com/en-us/sysinternals/downloads/sysinternals-suite
- **Topics covered:** Complete list of all Sysinternals tools, download links
- **Relevance:** Very High — these are essential support tools
- **Surprise factor for Linux users (1-5):** 3 — The depth and variety of tools exceeds standard Linux tooling
- **Estimated time:** Skim the list (~10 minutes), then learn tools individually

### 2. Process Explorer documentation
- **URL:** https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer
- **Topics covered:** Features, usage, keyboard shortcuts, troubleshooting scenarios
- **Relevance:** Very High — daily-use tool
- **Estimated time:** ~20 minutes read

### 3. Process Monitor documentation
- **URL:** https://learn.microsoft.com/en-us/sysinternals/downloads/procmon
- **Topics covered:** Features, filtering, boot logging, use cases
- **Relevance:** Very High — essential for deep troubleshooting
- **Estimated time:** ~25 minutes read

### 4. Autoruns documentation
- **URL:** https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns
- **Topics covered:** Features, what it monitors, comparison with Task Manager
- **Relevance:** High — useful for startup issues and malware investigation
- **Estimated time:** ~15 minutes read

### 5. TCPView documentation
- **URL:** https://learn.microsoft.com/en-us/sysinternals/downloads/tcpview
- **Topics covered:** Features, usage, network connection monitoring
- **Relevance:** Medium — useful for network troubleshooting
- **Estimated time:** ~10 minutes read

## PowerShell Resources

### 6. PowerShell for Administrators
- **URL:** https://learn.microsoft.com/en-us/powershell/scripting/learn/ps101/01-introduction
- **Topics covered:** PowerShell basics, cmdlets, pipeline, variables, scripting
- **Relevance:** High — PowerShell is the primary command-line administration tool
- **Surprise factor for Linux users (1-5):** 4 — Object-based pipeline, verb-noun naming, integrated help system
- **Estimated time:** ~2 hours for the full series
- **Notes:** Start with this series rather than trying to learn from scratch

### 7. PowerShell Event Log cmdlets
- **URL:** https://learn.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-eventlog
- **Topics covered:** Get-EventLog, Get-WinEvent, filtering, exporting events
- **Relevance:** High — essential for programmatic log analysis
- **Estimated time:** ~15 minutes read

### 8. Get-WindowsUpdateLog documentation
- **URL:** https://learn.microsoft.com/en-us/powershell/module/windowsupdate/get-windowsupdatelog
- **Topics covered:** Converting Windows Update logs to readable format
- **Relevance:** Medium — needed when troubleshooting Windows Update
- **Estimated time:** ~5 minutes read

## Additional Resources

### 9. Windows logging and diagnostics overview
- **URL:** https://learn.microsoft.com/en-us/windows/win32/etw/event-tracing-portal
- **Topics covered:** ETW (Event Tracing for Windows), trace logging, performance counters
- **Relevance:** Medium — advanced topic for deep troubleshooting
- **Estimated time:** Skim overview (~15 minutes)

### 10. NirSoft Utilities
- **URL:** https://www.nirsoft.net/
- **Topics covered:** 200+ small utility tools for Windows (BlueScreenView, NetworkView, CredentialsFileView, etc.)
- **Relevance:** High — many are extremely useful for support
- **Key tools:** BlueScreenView, BulletsPassView (saved passwords), CredentialsFileView, ProduKey (license keys), ShellExView (shell extensions)
- **Estimated time:** Browse as needed

### 11. Microsoft Message Analyzer (retired but archived)
- **URL:** https://learn.microsoft.com/en-us/message-analyzer/
- **Topics covered:** Network message capture and analysis
- **Relevance:** Low — retired tool but documentation still useful for understanding network diagnostics
- **Estimated time:** Skip unless specifically needed

### 12. Windows Performance Toolkit
- **URL:** https://learn.microsoft.com/en-us/windows-hardware/test/wpt/
- **Topics covered:** Windows Performance Recorder, Windows Performance Analyzer
- **Relevance:** Low — advanced performance analysis tool
